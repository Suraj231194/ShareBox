# PasteBox Project Overview

**PasteBox** is a secure, fast, and smart file-sharing platform designed for both casual and power users. It allows for instant file uploads and generation of shareable links or QR codes.

## Core Features
- **File Sharing**: Upload files securely and get a shareable link or QR code.
- **Storage**: Uses AWS S3 for reliable storage and MongoDB GridFS for large files.
- **Security**: Optional JWT-based authentication and secure download links.
- **User Dashboard**: Manage uploaded files, track downloads, and view history.
- **Temporary Files**: Option for expiration dates on shared files.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Redux Toolkit.
- **Backend**: Node.js, Express, MongoDB, AWS S3.

## Purpose
The goal is to provide a comprehensive, production-ready file-sharing solution that mimics real-world applications like WeTransfer or Google Drive sharing, but with a simplified and focused user experience.
